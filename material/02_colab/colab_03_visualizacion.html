Visualización de datos

matplotlib, seaborn, plotly e ipywidgets en Google Colab

Una visualización bien hecha hace en segundos lo que una tabla tarda minutos en comunicar. Para un docente, la diferencia entre un gráfico claro y uno confuso es la diferencia entre un estudiante que entiende el patrón y uno que se queda mirando números.

Este capítulo cubre tres niveles de visualización en Python: matplotlib para control total sobre cada elemento, seaborn para gráficos estadísticos con mínimo código, y plotly para visualizaciones interactivas que los estudiantes pueden explorar en el navegador. Cierra con ipywidgets para construir dashboards simples directamente en el notebook.

Todo el código corre directamente en Google Colab sin instalación adicional (excepto plotly, que requiere !pip install plotly en algunos entornos).

Configuración inicial del entorno de visualización

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import matplotlib.ticker as ticker
import seaborn as sns

# Configuración global de matplotlib
plt.rcParams.update({
    'figure.figsize':    (10, 6),
    'figure.dpi':         100,
    'axes.grid':          True,
    'grid.alpha':         0.3,
    'grid.linestyle':     '--',
    'axes.spines.top':    False,
    'axes.spines.right':  False,
    'font.size':          12,
    'axes.titlesize':     14,
    'axes.titleweight':   'bold',
    'axes.labelsize':     12,
    'xtick.labelsize':    10,
    'ytick.labelsize':    10,
    'legend.fontsize':    10,
    'legend.framealpha':  0.8,
})

# Tema seaborn
sns.set_theme(style='whitegrid', palette='husl', font_scale=1.1)

# Dataset base para todo el capítulo
rng = np.random.default_rng(seed=2025)
n = 300

carreras = ['Ing. Civil', 'Ing. Industrial', 'Ing. Eléctrica', 'Matemáticas']
semestres = ['2022-1', '2022-2', '2023-1', '2023-2', '2024-1', '2024-2']

df = pd.DataFrame({
    'carrera':      rng.choice(carreras, n, p=[0.35, 0.30, 0.20, 0.15]),
    'semestre':     rng.choice(semestres, n),
    'nota_final':   rng.normal(5.1, 1.1, n).clip(1, 7).round(1),
    'asistencia':   rng.beta(8, 2, n).round(2),
    'horas_estudio':rng.exponential(6, n).clip(0, 30).round(1),
    'aprobado':     None,  # se calcula abajo
    'genero':       rng.choice(['F', 'M'], n, p=[0.45, 0.55]),
})
df['aprobado'] = df['nota_final'] >= 4.0

print("✅ Entorno configurado. Dataset listo.")
print(df.head())

matplotlib: gráficos básicos

matplotlib es la base de toda la visualización en Python. Prácticamente todas las otras librerías (seaborn, pandas plot) construyen sobre ella. Entender su modelo de objetos te da control total sobre cualquier gráfico.

El modelo de objetos: Figure y Axes

# Forma explícita (recomendada — da control total)
fig, ax = plt.subplots(figsize=(10, 6))

# Forma implícita (más corta, menos control)
plt.figure(figsize=(10, 6))
plt.plot(...)  # trabaja sobre el "axes actual"

La forma explícita con fig, ax = plt.subplots() es la correcta para cualquier uso serio: sabes exactamente sobre qué Axes estás trabajando.

Gráfico de línea

# Evolución temporal de nota promedio por semestre
evolucion = df.groupby('semestre')['nota_final'].mean().reset_index()

fig, ax = plt.subplots(figsize=(10, 5))

ax.plot(
    evolucion['semestre'],
    evolucion['nota_final'],
    marker='o',          # marcador en cada punto
    markersize=8,
    linewidth=2,
    color='#2196F3',
    label='Nota promedio'
)

# Línea de referencia
ax.axhline(y=4.0, color='red', linestyle='--',
           linewidth=1.5, alpha=0.7, label='Nota mínima aprobación')

# Anotar cada punto con su valor
for _, row in evolucion.iterrows():
    ax.annotate(
        f"{row['nota_final']:.2f}",
        xy=(row['semestre'], row['nota_final']),
        xytext=(0, 12), textcoords='offset points',
        ha='center', fontsize=10, fontweight='bold'
    )

# Decorar
ax.set_title('Evolución del promedio de nota final por semestre')
ax.set_xlabel('Semestre')
ax.set_ylabel('Nota promedio')
ax.set_ylim(1, 7.5)
ax.legend()
ax.tick_params(axis='x', rotation=30)

plt.tight_layout()
plt.savefig('evolucion_notas.png', dpi=150, bbox_inches='tight')
plt.show()

Gráfico de dispersión (scatter)

fig, ax = plt.subplots(figsize=(10, 7))

colores_carrera = {c: plt.cm.tab10(i)
                   for i, c in enumerate(df['carrera'].unique())}

for carrera, grupo in df.groupby('carrera'):
    ax.scatter(
        grupo['asistencia'],
        grupo['nota_final'],
        label=carrera,
        color=colores_carrera[carrera],
        alpha=0.6,
        s=50,              # tamaño del marcador
        edgecolors='white',
        linewidths=0.5
    )

# Línea de tendencia (regresión lineal simple)
z = np.polyfit(df['asistencia'], df['nota_final'], 1)
p = np.poly1d(z)
x_line = np.linspace(df['asistencia'].min(), df['asistencia'].max(), 100)
ax.plot(x_line, p(x_line), 'k--', linewidth=1.5,
        alpha=0.7, label=f'Tendencia: y={z[0]:.1f}x+{z[1]:.1f}')

# Líneas de referencia
ax.axhline(y=4.0, color='red', linestyle=':', linewidth=1.5,
           alpha=0.6, label='Nota mínima')

ax.set_title('Relación entre asistencia y nota final')
ax.set_xlabel('Asistencia (proporción)')
ax.set_ylabel('Nota final')
ax.legend(loc='upper left', fontsize=9)

# Formatear eje x como porcentaje
ax.xaxis.set_major_formatter(ticker.PercentFormatter(xmax=1))

plt.tight_layout()
plt.show()

Mapas de calor (heatmap)

fig, axes = plt.subplots(1, 2, figsize=(16, 6))

# Heatmap de correlaciones numéricas
cols_numericas = ['nota_final', 'asistencia', 'horas_estudio',
                  'nota_final']  # en un caso real incluir más columnas
corr = df[['nota_final', 'asistencia', 'horas_estudio']].corr()

sns.heatmap(
    corr,
    annot=True,          # mostrar valores en cada celda
    fmt='.2f',           # 2 decimales
    cmap='coolwarm',     # paleta divergente (negativo=azul, positivo=rojo)
    center=0,            # centrar la escala de color en 0
    vmin=-1, vmax=1,
    square=True,
    linewidths=0.5,
    ax=axes[0]
)
axes[0].set_title('Mapa de correlaciones')

# Heatmap de tasa de aprobación (pivot)
pivot = df.pivot_table(
    index='carrera', columns='semestre',
    values='aprobado', aggfunc='mean'
) * 100

sns.heatmap(
    pivot,
    annot=True, fmt='.1f',
    cmap='RdYlGn',       # rojo=bajo, amarillo=medio, verde=alto
    vmin=50, vmax=100,
    linewidths=0.5,
    cbar_kws={'label': 'Tasa aprobación (%)'},
    ax=axes[1]
)
axes[1].set_title('Tasa de aprobación (%) — Carrera × Semestre')
axes[1].tick_params(axis='x', rotation=30)

plt.tight_layout()
plt.show()

# Pairplot: todas las relaciones entre variables numéricas
g = sns.pairplot(
    df[['nota_final', 'asistencia', 'horas_estudio', 'carrera']],

Tipos de gráfico y cuándo usar cada uno

La pregunta más importante en visualización no es “¿cómo hago este gráfico?” sino “¿qué gráfico debo usar?”.

Tabla de referencia rápida

Objetivo Gráfico recomendado Evitar
Ver distribución de una variable Histograma + KDE Pie chart
Comparar distribuciones entre grupos Boxplot o Violinplot Barras de media sola
Mostrar relación entre 2 numéricas Scatter Línea (si no hay orden natural)
Comparar promedios entre categorías Barras con IC Pie chart
Evolución temporal Línea Barras (si son muchos períodos)
Correlaciones entre muchas variables Heatmap Múltiples scatter individuales
Composición de un todo Barras apiladas % Pie con muchas categorías
Ranking Barras horizontales Barras verticales (difícil leer etiquetas largas)
AdvertenciaErrores visuales frecuentes en docencia
  • Eje Y que no empieza en 0 en gráficos de barras — exagera las diferencias visualmente.
  • Pie charts con más de 5 categorías — imposibles de leer. Usa barras.
  • Colores sin significado — usar rojo/verde para variables que no son bueno/malo.
  • Gráfico 3D — casi siempre confunde más que aclara.
  • Sin título, sin etiquetas de ejes — nunca publiques un gráfico sin contexto.
  • Demasiada información en un solo gráfico — divide en subplots si necesitas mostrar muchas cosas.

from pathlib import Path

def guardar_figura(fig, nombre, carpeta='graficos', formatos=('png', 'pdf')):
    """
    Guarda una figura en múltiples formatos en la carpeta de Drive del curso.

    Args:
        fig: objeto Figure de matplotlib
        nombre: nombre del archivo sin extensión
        carpeta: subcarpeta dentro del directorio del curso en Drive
        formatos: tupla de extensiones a exportar
    """
    CURSO = Path('/content/drive/MyDrive/mat281-2025-1')
    dir_salida = CURSO / carpeta
    dir_salida.mkdir(parents=True, exist_ok=True)

    for fmt in formatos:
        ruta = dir_salida / f"{nombre}.{fmt}"
        fig.savefig(
            ruta,
            dpi=300 if fmt == 'png' else None,
            bbox_inches='tight',
            facecolor='white' if fmt == 'png' else 'auto',
            format=fmt
        )
        print(f"✅ Guardado: {ruta}")

ipywidgets permite agregar controles interactivos (sliders, dropdowns, checkboxes) directamente en el notebook de Colab. El gráfico se actualiza en tiempo real cuando el usuario mueve un slider o cambia una selección.

def actualizar_dashboard(change=None):
    # Aplicar filtros
    df_filtrado = df.copy()

    if 'Todas' not in dropdown_carrera.value:
        df_filtrado = df_filtrado[df_filtrado['carrera'].isin(dropdown_carrera.value)]

    if 'Todos' not in dropdown_semestre.value:
        df_filtrado = df_filtrado[df_filtrado['semestre'].isin(dropdown_semestre.value)]

    if toggle_genero.value == 'Solo F':
        df_filtrado = df_filtrado[df_filtrado['genero'] == 'F']
    elif toggle_genero.value == 'Solo M':
        df_filtrado = df_filtrado[df_filtrado['genero'] == 'M']

    variable = radio_variable.value

    with output_dashboard:
        clear_output(wait=True)

        if len(df_filtrado) == 0:
            print("⚠️ Sin datos para los filtros seleccionados.")
            return

        fig, axes = plt.subplots(1, 3, figsize=(16, 5))
        fig.suptitle(
            f'Dashboard — {variable} | N={len(df_filtrado):,} registros',
            fontweight='bold', y=1.01
        )

        sns.histplot(data=df_filtrado, x=variable, kde=True,
                     bins=20, color='#2196F3', ax=axes[0])

        # Scatter vs asistencia (o nota si la variable es asistencia)

        sns.scatterplot(data=df_filtrado, x=otro, y=variable,
                        hue='carrera', palette='husl', alpha=0.6, s=40,

        axes[2].set_title(f'{variable} vs {otro}')

        plt.tight_layout()
        plt.show()

        print(f"\n📊 Resumen de {variable} por carrera:")
        print(df_filtrado.groupby('carrera')[variable]

Tipipywidgets en GitHub Pages / Quarto

Los widgets de ipywidgets funcionan en Colab pero no se renderizan en un sitio estático de GitHub Pages generado con Quarto. Para visualizaciones interactivas en el sitio del curso, usa plotly (genera HTML autocontenido) en lugar de ipywidgets.

Para dashboards más completos que se puedan publicar, considera Streamlit o Panel — ambos permiten aplicaciones web completas desde Python.

    df['asistencia'], df['nota_final'],
    c=df['aprobado'].astype(int),
    cmap='RdYlGn', vmin=0, vmax=1,
    alpha=0.5, s=30, edgecolors='none'
)

# Regresión lineal
z = np.polyfit(df['asistencia'], df['nota_final'], 1)
p = np.poly1d(z)
x_reg = np.linspace(df['asistencia'].min(), df['asistencia'].max(), 100)
ax4.plot(x_reg, p(x_reg), color=AZUL_UTFSM, linewidth=2,
         linestyle='-', label=f'Tendencia: y={z[0]:.1f}x+{z[1]:.1f}')

ax4.axhline(y=4.0, color=ROJO_UTFSM, linestyle='--',
            linewidth=1.5, alpha=0.7, label='Nota mínima')

ax4.set_xlabel('Asistencia (proporción)')
