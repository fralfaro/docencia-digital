Por qué importa la visualización en docencia

“Una visualización bien hecha hace en segundos lo que una tabla tarda minutos en comunicar.”

Para un docente, la diferencia entre un gráfico claro y uno confuso es la diferencia entre:

  • Un estudiante que entiende el patrón
  • Un estudiante que se queda mirando números

Tres niveles en Python:

Librería Para qué
matplotlib Control total sobre cada elemento
seaborn Gráficos estadísticos con mínimo código
plotly Visualizaciones interactivas en el navegador

Y ipywidgets para dashboards simples en el notebook.

Configuración global: buenas prácticas desde el inicio

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Configuración global de matplotlib
plt.rcParams.update({
    'figure.figsize':    (10, 6),
    'figure.dpi':         100,
    'axes.grid':          True,
    'grid.alpha':         0.3,
    'grid.linestyle':     '--',
    'axes.spines.top':    False,
    'axes.spines.right':  False,
    'font.family':        'sans-serif',
})

# Estilo de seaborn
sns.set_theme(style='whitegrid', palette='muted')

Hacerlo una vez, al inicio del notebook

Esta celda establece el estilo para todos los gráficos del notebook. Los estudiantes no tienen que preocuparse por el formato en cada gráfico individual.

matplotlib: los gráficos más comunes

Dispersión (scatter):

fig, ax = plt.subplots()
ax.scatter(df['x'], df['y'],
           c='steelblue', alpha=0.6, s=40)
ax.set_xlabel('Variable X')
ax.set_ylabel('Variable Y')
ax.set_title('Relación entre X e Y')
plt.tight_layout()
plt.show()

Línea temporal:

ax.plot(df['fecha'], df['valor'],
        color='steelblue', linewidth=2,
        label='Precio')
ax.legend()

Histograma:

ax.hist(df['nota'], bins=20,
        color='steelblue', edgecolor='white',
        alpha=0.8)
ax.axvline(df['nota'].mean(),
           color='red', linestyle='--',
           label='Media')

Barras:

categorias = df.groupby('carrera')['nota'].mean()
ax.bar(categorias.index, categorias.values,
       color='steelblue')
ax.tick_params(axis='x', rotation=45)

fig, ax = plt.subplots()

Esta es la sintaxis orientada a objetos. Es más flexible que plt.plot() directamente, especialmente al hacer múltiples subgráficos.

Múltiples subgráficos

fig, axes = plt.subplots(1, 3, figsize=(15, 5))

# Subgráfico 1: histograma
axes[0].hist(df['nota'], bins=15, color='steelblue')
axes[0].set_title('Distribución de notas')

# Subgráfico 2: boxplot
axes[1].boxplot(df['nota'])
axes[1].set_title('Boxplot de notas')

# Subgráfico 3: scatter
axes[2].scatter(df['horas_estudio'], df['nota'], alpha=0.5)
axes[2].set_xlabel('Horas de estudio')
axes[2].set_ylabel('Nota')
axes[2].set_title('Estudio vs Nota')

plt.suptitle('Análisis de rendimiento académico', fontsize=14)
plt.tight_layout()
plt.show()

plt.tight_layout() siempre al final

Ajusta automáticamente el espaciado entre subgráficos para que los títulos y etiquetas no se superpongan.

seaborn: gráficos estadísticos con mínimo código

Distribuciones:

# Histograma + KDE
sns.histplot(df['nota'], kde=True, bins=20)

# Boxplot por categoría
sns.boxplot(x='carrera', y='nota', data=df)

# Violín (distribución + cuartiles)
sns.violinplot(x='carrera', y='nota', data=df)

Relaciones:

# Scatter con regresión
sns.regplot(x='horas', y='nota', data=df)

# Scatter por categoría
sns.scatterplot(x='horas', y='nota',
               hue='carrera', data=df)

Calor y correlaciones:

# Mapa de calor de correlaciones
corr = df.select_dtypes('number').corr()
sns.heatmap(corr, annot=True, fmt='.2f',
            cmap='coolwarm', center=0)

Matriz de dispersión:

sns.pairplot(df, hue='carrera',
             vars=['nota', 'horas', 'asistencia'])

seaborn para exploración rápida

Una línea de seaborn equivale a 15 líneas de matplotlib. Úsalo para exploración; usa matplotlib cuando necesites control fino.

Cuándo usar plotly

Cuando los estudiantes necesitan explorar los datos. El hover para ver valores individuales es muy valioso en datasets grandes.

ipywidgets: dashboards interactivos en el notebook

import ipywidgets as widgets
from IPython.display import display

# Slider para el número de bins
bins_slider = widgets.IntSlider(
    value=20, min=5, max=100,
    description='Bins:'
)

# Dropdown para elegir la variable
var_dropdown = widgets.Dropdown(
    options=['nota', 'horas', 'asistencia'],
    value='nota',
    description='Variable:'
)

def actualizar(bins, variable):
    plt.figure(figsize=(8, 4))
    plt.hist(df[variable], bins=bins,
             color='steelblue', edgecolor='white')
    plt.title(f'Distribución de {variable}')
    plt.show()

widgets.interactive(actualizar,
    bins=bins_slider,
    variable=var_dropdown)

Widgets más útiles para docencia:

Widget Uso típico
IntSlider Ajustar k en k-NN, número de árboles
FloatSlider Tasa de aprendizaje, umbral
Dropdown Elegir dataset o variable
Checkbox Activar/desactivar opciones
RadioButtons Seleccionar tipo de modelo

Intuición sin código

Los estudiantes desarrollan intuición sobre parámetros (overfitting, underfitting, k) sin tener que reescribir código en cada experimento.

Guardar gráficos en alta resolución

Guardar con matplotlib:

fig, ax = plt.subplots(figsize=(10, 6))
ax.scatter(df['x'], df['y'])
ax.set_title('Mi gráfico')

# PNG de alta resolución
fig.savefig('grafico.png', dpi=300,
            bbox_inches='tight')

# PDF vectorial (para LaTeX o Word)
fig.savefig('grafico.pdf', bbox_inches='tight')

# SVG (escalable para web)
fig.savefig('grafico.svg', bbox_inches='tight')

Descargar desde Colab:

from google.colab import files
files.download('grafico.png')

Guardar plotly como HTML:

# HTML interactivo (con toda la interactividad)
fig.write_html('grafico.html')

# PNG estático (requiere kaleido)
!pip install kaleido -q
fig.write_image('grafico.png', scale=3)

Para el sitio del curso

  • PNG (dpi=150) → notebooks e informes
  • SVG → sitio Quarto (escalable, sin pixelado)
  • HTML → plotly embebido en el sitio interactivo

Guía de selección: ¿qué gráfico para cada situación?

Pregunta Gráfico recomendado
¿Cómo se distribuye una variable numérica? Histograma + KDE (seaborn)
¿Hay outliers? Boxplot / Violín
¿Relación entre dos variables numéricas? Scatter (matplotlib)
¿Comparar categorías? Barras (matplotlib)
¿Evolución en el tiempo? Línea (matplotlib)
¿Correlación entre variables? Heatmap (seaborn)
¿Explorar dataset grande interactivamente? Scatter plotly
¿Distribución por grupos? Violín o boxplot (seaborn, hue=)

La regla de oro

El mejor gráfico es el más simple que responde la pregunta. No agregues dimensiones (colores, tamaños, facetas) que no corresponden a variables relevantes.
