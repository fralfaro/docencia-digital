pandas: la hoja de cálculo programable

¿Qué es pandas?

La librería central del ecosistema de ciencia de datos en Python.

  • NumPy = la calculadora
  • pandas = la hoja de cálculo programable

Permite cargar, explorar, limpiar, transformar y exportar datos tabulares con sintaxis expresiva y eficiente.

import pandas as pd

# Verificar versión
print(pd.__version__)  # 2.2.x

El DataFrame:

Una tabla bidimensional con filas y columnas etiquetadas.

Cargar datos: CSV, Excel, URL, Google Drive

Desde CSV local o URL:

# Desde URL pública (recomendado para docencia)
df = pd.read_csv('https://raw.githubusercontent.com/...')

# CSV local
df = pd.read_csv('datos.csv')

# Con opciones
df = pd.read_csv('datos.csv',
    sep=';',           # separador punto y coma
    encoding='utf-8',
    decimal=',')       # decimales con coma (España/Chile)

Desde Excel:

df = pd.read_excel('datos.xlsx', sheet_name='Hoja1')

Desde Google Drive en Colab:

from google.colab import drive
drive.mount('/content/drive')

df = pd.read_csv(
    '/content/drive/MyDrive/datos.csv'
)

Formatos adicionales:

pd.read_json('datos.json')
pd.read_parquet('datos.parquet')
pd.read_sql(query, conexion)

La secuencia de exploración:

Paso Función Pregunta que responde

Buena práctica docente

Muestra esta secuencia al inicio de cada notebook de análisis. Los estudiantes la internalizan como ritual.

Selección y filtrado

Seleccionar columnas:

# Una columna → Serie
df['nota']

# Varias columnas → DataFrame
df[['nombre', 'nota', 'carrera']]

Filtrar filas (condiciones):

# Nota >= 4
aprobados = df[df['nota'] >= 4]

# Múltiples condiciones
civil_buenos = df[
    (df['carrera'] == 'Civil') &
    (df['nota'] >= 5.5)
]

# Valores en una lista
carreras = df[df['carrera'].isin(['Civil', 'Industrial'])]

Selección por posición y por etiqueta:

# Por posición (iloc)
df.iloc[0]      # primera fila
df.iloc[0:5]    # filas 0 a 4
df.iloc[:, 1:3] # columnas 1 a 2

# Por etiqueta (loc)
df.loc[0]             # índice 0
df.loc[:, 'nota']     # toda la columna nota
df.loc[df['nota'] > 5, ['nombre', 'nota']]

loc vs iloc

loc = por etiqueta (nombre de la columna, valor del índice)

iloc = por posición (número entero, como una lista de Python)

Limpieza de datos: los casos más frecuentes

Manejar valores nulos:

# Eliminar filas con cualquier nulo
df_limpio = df.dropna()

# Eliminar solo si hay nulo en columnas específicas
df_limpio = df.dropna(subset=['nota', 'carrera'])

# Rellenar con un valor
df['nota'].fillna(df['nota'].mean(), inplace=True)

# Rellenar con método
df['nota'].fillna(method='ffill', inplace=True)

Duplicados:

df.duplicated().sum()        # cuántos hay
df = df.drop_duplicates()   # eliminar

Tipos de datos:

# Convertir columna a número
df['nota'] = pd.to_numeric(df['nota'], errors='coerce')

# Convertir a fecha
df['fecha'] = pd.to_datetime(df['fecha'])

# Convertir a categoría (eficiente para texto repetido)
df['carrera'] = df['carrera'].astype('category')

Renombrar columnas:

df = df.rename(columns={
    'nota_final': 'nota',
    'nombre_alumno': 'nombre'
})

Transformaciones: apply, map y asignación

Crear columnas nuevas:

# Operación directa
df['nota_ponderada'] = df['nota'] * 0.7 + df['extra'] * 0.3

# Función condicional con apply
def letra(nota):
    if nota >= 5.5: return 'A'
    elif nota >= 4.0: return 'B'
    else: return 'F'

df['letra'] = df['nota'].apply(letra)

# Con lambda (funciones cortas)
df['aprobado'] = df['nota'].apply(lambda x: x >= 4.0)

Operaciones de texto:

# Minúsculas, strip, reemplazar
df['nombre'] = df['nombre'].str.lower().str.strip()
df['email'] = df['email'].str.replace('@correo.cl', '@usm.cl')

# Extraer parte de texto
df['apellido'] = df['nombre'].str.split(' ').str[-1]

apply vs operaciones vectorizadas

Cuando puedes, usa operaciones de pandas directas (más rápidas). Usa apply cuando necesitas lógica compleja que no se puede vectorizar.

Agrupación y agregación: groupby

Pregunta típica: “¿Cuál es la nota promedio por carrera?”

# Promedio de nota por carrera
df.groupby('carrera')['nota'].mean()

# Múltiples métricas
resumen = df.groupby('carrera').agg(
    promedio=('nota', 'mean'),
    minimo=('nota', 'min'),
    maximo=('nota', 'max'),
    conteo=('nota', 'count')
)

# Dos niveles de agrupación
df.groupby(['carrera', 'semestre'])['nota'].mean()

Pivot tables:

tabla = df.pivot_table(
    values='nota',
    index='carrera',
    columns='semestre',
    aggfunc='mean'
)

groupby + reset_index()

Después de un groupby, el resultado tiene el grupo como índice. Usa .reset_index() para convertirlo en DataFrame normal y poder graficar directamente.

promedios = df.groupby('carrera')['nota'].mean().reset_index()

Merge y concatenación de DataFrames

Merge (equivalente a JOIN en SQL):

# Inner join (solo filas que coinciden en ambos)
df_merge = pd.merge(df_notas, df_estudiantes,
                    on='id_alumno', how='inner')

# Left join (todas las filas de df_notas)
df_merge = pd.merge(df_notas, df_estudiantes,
                    on='id_alumno', how='left')

# Columnas con nombres distintos
df_merge = pd.merge(df_a, df_b,
                    left_on='id', right_on='alumno_id')

Concatenación (apilar DataFrames):

# Apilar verticalmente (mismo formato)
df_total = pd.concat([df_2024, df_2025],
                     ignore_index=True)

# Agregar columna para identificar origen
df_2024['año'] = 2024
df_2025['año'] = 2025
df_total = pd.concat([df_2024, df_2025])

merge vs concat

merge = unir por columna clave (como JOIN)
concat = apilar DataFrames (como UNION ALL)

df.to_parquet('resultados.parquet')

resumen = df.groupby('carrera').agg(...)
